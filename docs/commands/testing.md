# Testing

`rome test internal/compiler/lint/rules/harness.test.ts --update-snapshots`

Running the test runner:

`./rome test`

Running a file:

`./rome test path/to/files`

Updating snapshots:

`./rome test --update-snapshots`

Enabling logging:

`./rome test --no-suppress-logs`
